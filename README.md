# Pharo-MessageFlowBrowser

A message flow browser for Pharo written by T. Bergmann (Astares) in 2018 / 2019.

# Quick Start
## Installation
```Smalltalk
Metacello new 
	repository: 'github://astares/Pharo-MessageFlowBrowser/src';
	baseline: 'MessageFlowBrowser';
	load
```
